# Simple HTTP Client

A lightweight CLI HTTP client for academic purposes.

## Features

  - HTTP 0.9, HTTP 1.0, HTTP 1.1 and HTTP 2.0 
  - Based on popular RFC

To run:
```
$ ./client.c
```
